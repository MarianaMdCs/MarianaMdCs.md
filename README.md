- `NestJS` `TypeORM` `PostgreSQL` `JWT` `REST API`

- `Next.js` `React Native` `Express` `MongoDB` `JWT`

### 💬 [Real-Time Chat App]([https://github.com/MarianaMdCs/[repositorio]](https://github.com/MarianaMdCs/Csharp-RealTime-Chat)
